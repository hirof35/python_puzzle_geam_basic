# 15 Puzzle (Pygame Edition)

Pythonの外部ライブラリ `Pygame` を利用して開発した、クラシックな15パズルゲームです。グリッドベースの座標管理と、タイル移動のロジックを直感的に実装しています。

## 概要
4x4のグリッド上に配置された1から15までの数字タイルを、マウス操作によって空白スペース（None）へ動かし、数字を順番に並べるパズルゲームです。

## 主な機能と実装の工夫
*   **動的グリッド生成**: 
    リストの内包表記と `random.shuffle` を組み合わせ、実行ごとに異なるランダムな初期配置を生成します。
*   **直感的な操作性 (Man-Machine Interface)**: 
    マウスのクリック座標からタイル座標（0〜3）への数学的な変換（`pos // TILE_SIZE`）を行い、直感的なクリック操作を実現しています。
*   **移動判定ロジック**: 
    クリックされたタイルが「空白マスの上下左右1マス以内にあるか」を `abs(x - ex) == 1` のような絶対値計算で判定。数学的な簡潔さと堅牢さを両立させています。
*   **スワップアルゴリズム**: 
    Python特有の変数入れ替え（`a, b = b, a`）を活用し、一時変数を使わずに効率的なタイルの位置交換を行っています。

## 動作環境
*   **言語**: Python 3.x
*   **ライブラリ**: Pygame

## 導入・実行手順
1.  **ライブラリのインストール**
    ```bash
    pip install pygame
    ```
2.  **実行**
    ```bash
    python main.py
    ```

## 操作方法
*   **タイルをクリック**: 空白（白背景）に隣接する数字タイルをクリックすると、その位置へ移動します。
*   **終了**: ウィンドウのクローズボタン（×）を押すと終了します。

## システム構成
*   **描画ループ**: Pygameのメインループ（`while running`）によるリアルタイム描画。
*   **座標管理**: 2次元リスト（`board[y][x]`）によるタイルの状態管理。
*   **タイルサイズ**: 400x400の画面を等分した100px単位のタイル。
